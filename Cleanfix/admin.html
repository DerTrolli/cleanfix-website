Export / Import

Da die Daten lokal gespeichert werden, sind sie gerätespezifisch. Mit dem Export können Sie alle Einstellungen als JSON-Datei sichern und auf einem anderen Gerät importieren.

Cache leeren

Setzt alle gespeicherten Änderungen zurück. Die Website zeigt dann wieder die Standard-Inhalte aus dem Code. Diese Aktion kann nicht rückgängig gemacht werden.
